# Phase 3 Status

## 구현 완료

- URL 정규화 후 정확 일치 중복 제거
- 제목 정규화와 보수적 유사도 임계값 `0.92`
- 최초 수집 기사를 대표 기사로 유지
- 통합된 기사 출처 목록 보존

## 검증 완료

- 추적 파라미터·fragment가 다른 동일 URL 통합 테스트
- 문장부호만 다른 유사 제목 통합 테스트
- 서로 다른 제목을 유지하는 오탐 방지 테스트

## 남은 완료 기준

- 실제 수집 기사 평가 데이터셋 작성
- 실제 데이터 기준 오탐·누락 측정 및 임계값 조정

실제 데이터 검증 전에는 오탐을 줄이기 위해 높은 유사도 임계값을 사용한다.
