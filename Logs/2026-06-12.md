## 특이사항

- 실제 외부 뉴스 수집, OpenAI API 호출, Telegram 발송은 승인 없이 실행하지 않음
- 실제 OpenClaw 환경 설정과 호출은 승인 없이 실행하지 않음
- PostgreSQL V2 migration은 승인 후 적용했으며 기존 기사 행은 보존됨
- PostgreSQL V3 적용 시 Backend와 함께 Worker 컨테이너도 재생성되었으나 외부 동작은 비활성 상태 유지
- 현재 PostgreSQL 기사 수가 0건이므로 실데이터 기준 p95와 장기 성능 추세는 측정하지 못함
- 운영 추세 스냅샷 기준선은 측정 경로 검증용이며 실데이터 품질·성능 완료 판정에는 사용하지 않음
- 실제 외부 모니터링 수신 시스템과 webhook 전송은 승인 및 대상 정보가 없어 수행하지 않음
- PostgreSQL 16은 Spring Boot 2.7 내장 Flyway 8.5 공식 검증 범위 밖 경고가 있으나 migration은 성공함
- 로컬 Maven은 JDK 18에서 실행하지만 Java 8 대상 바이트코드를 생성하며, Docker Backend는 Java 8로 빌드·실행함
